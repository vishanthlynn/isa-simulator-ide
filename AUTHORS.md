This project was designed and developed by Vishanth Dandu as a comprehensive demonstration of:

- Custom Instruction Set Architecture (ISA) design
- Instruction set simulator implementation in Python
- Full-stack web application development (React + FastAPI)
- Real-time WebSocket communication
- Interactive debugging and visualization tools
- Assembly language assembler implementation

## Project Overview

The ISA Simulator + Web IDE project showcases:
- A custom 16-bit instruction set with multiple instruction formats
- A complete simulator with fetch-decode-execute cycle
- A web-based IDE with syntax highlighting and real-time visualization
- Register file, memory, and instruction trace visualization
- Step-by-step debugging capabilities
